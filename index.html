What is Unicode Hindi Text?

Unicode is the international standard for encoding text across all languages and scripts, including Hindi (Devanagari). When you type Hindi in modern applications — whether in MS Word using the Mangal font, on a smartphone, on a website, or in Google Docs — the underlying text is almost certainly stored in Unicode format.

Unicode Hindi is portable and universally supported. You can share it via email, display it on websites, and open it on any device without worrying about font compatibility. However, older software systems and some printing workflows were built around non-Unicode fonts like Krutidev, which is why conversion is often needed.

What is Krutidev Font?

Krutidev is one of the oldest and most recognisable Hindi fonts in India. Developed well before Unicode became mainstream, it uses a proprietary ASCII-based encoding where each keyboard key maps to a specific Devanagari character or matra. This made it very convenient for typists before Unicode keyboards were widely available.

Even today, Krutidev remains deeply embedded in Indian workflows. Government typing exams (including CPCT, High Court, and various state recruitment tests) are often conducted in Krutidev. Many printing presses, old publishing houses, and district-level offices still operate on Krutidev. So if you're working with any of those systems, knowing how to convert Unicode to Krutidev is genuinely useful.

Why Can't You Just Copy-Paste Between Unicode and Krutidev?

This is one of the most common questions people have. The short answer: Unicode and Krutidev use completely different character encoding systems.

In Unicode, each Hindi character has a unique code point assigned by the Unicode Consortium — for example, क is U+0915. In Krutidev, that same visual character is represented by a plain ASCII character that looks completely different when viewed without the Krutidev font applied. So if you simply copy Unicode Hindi text and paste it into a Krutidev environment, you'll see garbled symbols instead of Hindi.
